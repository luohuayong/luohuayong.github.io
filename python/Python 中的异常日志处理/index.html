Python 中的异常日志处理

发表于 2023-09-04 更新于 2023-09-12 阅读次数：

下面的代码中，包含 3 种使用日志记录代码异常的方式（当然还有更多方法，但这篇文章的重点是对比下面三种方式）

logger.error(e)

logger.error(e) 会按给定格式输出错误日志, 但是默认不包含回溯信息.

logger.error(e,stack_info=True,exc_info=True)

logger.error(e,stack_info=True,exc_info=True) 除了按格式输出了错误日志外，还包含回溯和堆栈信息。

logger.exception (e)

最后 logger.exception (e) 将给出该异常所在行和回溯信息，而无需设置任何标志。它不会给出堆栈信息，除非在参数中设置 stack_info=True。
