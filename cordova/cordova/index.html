Cordova 从入门到放弃

发表于 2023-08-12

Cordova 是 Apache 基金会下的一个开源的移动开发框架。它允许你使用标准 Web 技术 - HTML5、CSS3 和 JavaScript 用于跨平台开发。应用程序在每个平台对应的包装器中执行，并依靠设备的标准 API 来访问设备的功能，例如传感器、数据、网络状态等。

快速启动

若要生成和运行应用，需要为对应的平台安装 SDK。浏览器开发，则不需要任何平台 SDK。

安卓平台

注意: cordova-android 10 以上版本必须安装 JDK 11 , 使用 JDK 8 构建 android 客户端时会失败.

安装 Android Studio

官方下载地址

https://developer.android.com/studio

安装 Java JDK

甲骨文官方下载需要注册用户,太繁琐. 直接在第三方下载 JDK 安装包

http://www.codebaoku.com/jdk/jdk-index.html

配置 Java 开发环境

https://www.runoob.com/java/java-environment-setup.html

安装 Gradle

Gradle 是一个开源的自动化构建工具. 官网地址

https://gradle.org/

  • 下载最新版 zip 文件
  • 解压后放在 C:\Gradle\ 下
  • 配置环境变量,在 Path 下添加路径 C:\Gradle\gradle-8.2.1\bin

运行

生成的 android apk 文件路径
<MyApp>\platforms\android\app\build\outputs\apk\debug\app-debug.apk

生成的 electron 文件路径
<MyApp>\platforms\electron\build\HelloCordova Setup 1.0.0.exe

平台支持

Windows 下开发, Cordova 可以生成 android / windows / electron 应用, 不能生成 iOS / OSX 应用.
Mac 下开发, Cordova 可以生成 android / iOS / OSX / electron 应用, 不能生成 windows 应用.

插件

插件是将本地组件包装为 JavaScript 接口,以允许应用访问本机设备功能。

组件名 说明
cordova-plugin-battery-status 电池状态
cordova-plugin-camera 照相机
cordova-plugin-device 设备
cordova-plugin-dialogs 对话框
cordova-plugin-file 文件
cordova-plugin-geolocation 地理位置
cordova-plugin-inappbrowser 应用内浏览器
cordova-plugin-media 媒体
cordova-plugin-media-capture 媒体捕获
cordova-plugin-network-information 网络信息
cordova-plugin-screen-orientation 屏幕方向
cordova-plugin-splashscreen 初始屏幕
cordova-plugin-statusbar 状态栏
cordova-plugin-vibration 振动

关联框架

Capacitor 和 Cordova 都管理 Web 视图，并提供一种结构化的方式来向 Web 代码公开本机功能。
Capacitor 被认为是 Cordova 的修改版本, Capacitor 支持现代渐进式 Web 应用（PWA）程序开发。这使用户的应用程序与不断发展的 Web 平台保持一致，同时在支持它的平台上启用强大的本机设备功能。
