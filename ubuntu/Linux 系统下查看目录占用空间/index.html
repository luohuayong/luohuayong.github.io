Linux 系统下查看目录占用空间

发表于 2023-09-24 阅读次数：

在 Linux 系统中,我们可以使用 du 命令来查看目录的占用空间.

du 命令使用格式为: du [选项] [目录]

其中, 常用选项包括:

  • -h: 以可读的方式显示尺寸,会自动以合适的单位显示尺寸 (G/M/K)
  • -B: 以指定单位显示尺寸, 比如 -BK -BM -BG
  • -d: 目录深度, 默认递归所以目录 -d 1 只统计一级子目录

如果不指定目录, 则统计当前目录.
