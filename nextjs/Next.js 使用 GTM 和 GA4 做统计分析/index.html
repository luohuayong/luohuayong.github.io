Next.js 使用 GTM 和 GA4 做统计分析

发表于 2023-09-15 阅读次数：

GTM 全称是 Google Tag Manager（谷歌标签管理器）, GA4 是 Google Analytics（谷歌分析）的最新版本.
这两个都是由 Google 提供的免费统计分析工具.

Google Analytics 用于分析和报告网站数据，而 Google Tag Manager 用于管理和部署各种标签和代码，包括 Google Analytics 跟踪代码。

GTM 账号

GA 的访问地址 https://analytics.google.com/
GTM 访问地址 https://tagmanager.google.com/

谷歌账号登陆后,可以直接使用.

现在 GA 中创建媒体资源,媒体资源下创建数据流,在管理界面能查看到媒体资源的衡量 ID,这个就是 GA id
然后在 GTM 中创建账号,账号下创建容器,容器类型选择Google 代码,代码 ID 配置为前面的 GA id 触发条件选择All Page,在账号列表下能看到容器 ID, 这个容器 ID 就是 GTM id

Next.js 中使用 GTM

在 Next.js 中使用 GTM 可以通过 react-gtm-module 包进行配置.
首先安装 react-gtm-module 包

然后在 \_app.tsx 中进行配置,注意将 gtmId 更换为自己的 GTM id
