从 Outline VPN 的访问密钥中获取访问信息

发表于 2023-07-06 更新于 2023-08-12 阅读次数：

可以从 outline vpn 现有访问密钥中提取访问信息。下面的示例显示了在哪里可以找到每个相关信息：“服务器”、“服务器端口”、“密码” 和 “方法”。

静态访问密钥示例

ss://Y2hhY2hhMjAtaWV0Zi1wb2x5MTMwNTo1UkVmeFRqbHR6Mkw@outline-server.example.com:17178/?outline=1

服务器

静态访问密钥中 “@” 和 “:” 符号之间的部分就是 “服务器”

ss://Y2hhY2hhMjAtaWV0Zi1wb2x5MTMwNTo1UkVmeFRqbHR6Mkw@outline-server.example.com:17178/?outline=1

服务器端口
静态访问密钥中 第二个“:”符号和后面的“/”符号之间的部分就是 “服务器端口”

ss://Y2hhY2hhMjAtaWV0Zi1wb2x5MTMwNTo1UkVmeFRqbHR6Mkw@outline-server.example.com:17178/?outline=1

密码
这里会有些复杂。静态访问密钥中 介于 “ss://” 和 “@”之间的部分就是以 base64 格式编码的 “密码”。

ss://Y2hhY2hhMjAtaWV0Zi1wb2x5MTMwNTo1UkVmeFRqbHR6Mkw@outline-server.example.com:17178/?outline=1

在我们的例子中，当我们对密钥的这一部分进行解码时，我们得到下面的结果:

chacha20-ietf-poly1305:5REfxTjltz2L

其中粗体部分（在“:”之后）是“密码”

方法
而在上述结果中“:”之前的部分是“方法”：

chacha20-ietf-poly1305:5REfxTjltz2L

总而言之，我们这里的示例键的最终访问信息 JSON 如下所示:

# outline vpn # 密钥
